# pitch

Présentation GitPitch basée sur `PITCHME.md` et `PITCHME.yaml`.

## Aperçu local

GitPitch étant désormais arrêté, vous pouvez prévisualiser le contenu en ouvrant `PITCHME.md` dans votre éditeur Markdown ou en utilisant un serveur Reveal.js local si besoin.

## Structure

- `PITCHME.md` : contenu des slides
- `PITCHME.yaml` : configuration (thème, transition, plugins)
